## 更改

- 更新了矛使取消骑乘状态的速度阈值以匹配基岩版
- 修复了幼年美西螈右后腿不会在游泳时正确移动的问题
- 幼年猫现在变大了以匹配幼年豹猫的大小
- 幼年马模型不再略微变大    

### UI

- 增加了一个新的显示选项名为“独占全屏”，用于控制在全屏模式下是否应该完全控制该显示器
  - 默认为 `false` (即非独占模式)
  - 仅当游戏重启后更改才会生效
  - 警告：独占全屏模式可能会阻止 IME 输入法编辑器的使用

### 交易

- 大师级图书管理员现在会有三项可用的交易，以确保在使用“村民交易的平衡性调整”实验性内容时总能提供一个附魔书交易

## 技术性更改

### 文本组件

- 在服务器状态消息（MOTD）的嵌套超过 16 次的组件现在会发起嵌套并使用省略号代替
