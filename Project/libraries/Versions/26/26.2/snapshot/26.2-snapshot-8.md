## 新特性

### 硫磺洞穴生物群系

- 如果暴露在地表，群系会生成所有地下硫磺洞穴生物群系的地物

### 硫磺史莱姆

- 吸收了方块的硫磺史莱姆现在会免疫演岩浆块的伤害，以及免疫其他吸收了岩浆块的硫磺史莱姆的伤害

## 更改

- 游戏现在会使用 OpenGL 作为默认图形 API
- 为了匹配基岩版，床现在略微更弹了

### 对 “图形 API” 视频选项的更改

- “Vulkan” 现在被标记为 “试验性”
  - 如果想要的话你仍可以使用它，其在大部分常见系统上运行良好，但是在一些其它系统上可能会降低性能或导致不稳定
- “默认” 选项现在会使用 OpenGL
- 所有玩家的这个选项会都会被重置为 “默认”

### UI 精灵图

- 增加了 `friends/background_dark` — `friends/background` 面板精灵图的深色变种，用于好友 UI 的确认对话框的背景

## 技术性更改

- 数据包版本现在为106.0

### 伤害类型

- 增加了新的 `minecraft:sulfur_cube_hot` 伤害类型

## 数据包版本 106.0

### 命令

#### Publish

- `publish` 现在有了一个可选的布尔值作为其第一个参数，用于指定服务器是否应该开放到在线多人游戏，或仅是局域网多人游戏
  - 默认值：true （在线多人游戏）

#### Unpublish

- 增加了 `unpublish` 命令，停止公开你的集成服务器（若有）
