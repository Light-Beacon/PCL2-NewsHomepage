今天为大家带来 26.2 预发布版 5，本次对极限模式设置进行了更改，并包含其他修复！正如《我的世界》全球嘉年华所宣布的，更新将于 6 月 16 日正式发布！
祝各位（谨慎）愉快地在极限模式下探索！

## 更改

- 若启动时检测到崩溃，图形 API 的游戏设置现在会自动变更
- 现在硫方怪的“弹起”音效会正确播放
- 若作为世界的主机，使用 F3 + F4 或命令更改游戏模式后，现在会同步更改世界的默认游戏模式
- 在极限模式世界中，“游戏模式”和“允许命令”按钮将不再可用

## 图形 API 的启动与回退机制

- 若游戏因设置了“优先使用 Vulkan”而于启动时崩溃，该设置将自动改为“默认”
- 设为“默认”时，游戏在启动时仍会尝试收集关于 Vulkan 的系统信息
- 若在设置为“默认”时游戏启动崩溃，该设置将自动改为“优先使用 OpenGL”
- 设为“优先使用 OpenGL”时，游戏将不会与 Vulkan 进行任何交互，以防引发崩溃

## 获取预发布版

预发布版适用于《我的世界：Java 版》。要安装预发布版，请打开[![《我的世界》启动器(https://www.minecraft.net/content/minecraft-net/language-masters/download)]，并在“安装”选项卡中启用快照版本。

测试版本可能会损坏你的世界，请备份存档，或将其与主要世界存放在不同文件夹中运行。
