## 更改

- 在游戏启动时检测到崩溃会更改图形 API 游戏设置
- 在击中“弹性”原型的硫方怪现在被击打时会播放正确的音效
- 当世界的主机当使用 F3+F4 或使用命令更改游戏模式时现在会更改世界的默认游戏模式
- “游戏模式”和“允许命令”按钮在极限模式世界中不再可用

### 图形 API 在游戏启动的行为和回退行为

- 如果游戏启动时在使用 “优先使用Vulkan”，设置会自动更改为“默认”
- 当设置为“默认”，游戏仍会在启动时尝试收集有关 Vulkan 的系统信息
- 如果游戏启动时在使用 “默认”，设置会自动更改为“优先使用OpenGL” 
  - 当使用“优先使用OpenGL”，在这种情况的崩溃不会产生与 Vulkan 之间的交互
