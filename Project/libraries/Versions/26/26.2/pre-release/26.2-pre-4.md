## 更改

- 当游戏被粒子占满，不再会移除正在飞行的粒子，而会随机选择粒子是否渲染以保持数量低于粒子限制
  - 这项更改尤其是在大量的间歇泉喷发时可见

### 硫方怪

- 硫方怪现在会随着击退附魔等级的提升击退到更远的距离

### 烈性硫磺

- 末影龙不再会被间歇泉影响

### UI

- “多人游戏选项”已经重命名回了“开放到局域网”
- 修复了一个 Realms 配置屏幕可能打开失败的问题

#### 世界选项屏幕

- 世界的主机现在可以在世界选项屏幕中访问“游戏模式”和“允许命令”按钮

#### 开放到局域网屏幕

- 不同于世界选项，世界的主机可以为其它玩家配置“游戏模式”和“允许命令”的值
  - 这些按钮可以在“Settings for Other Players”标题下找到

## 技术性更改

- 数据包版本现在为 107.1
- `particle_render_stats` 现在包含了不同粒子组的计数（`SQ` 为 `single quads`， `NR` 为 `no render`， `EG` 为 `elder guardians` 以及 `IP` 为 `item pickup`），和总数类似。

## 数据包版本 107.1

### 实体标签

- 增加了 `#not_affected_by_geysers` - 不会收到间歇泉喷发的上升冲量影响
