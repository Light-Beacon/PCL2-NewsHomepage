## 更改

### 纹理过滤

- 视频设置中的新选项
- 控制以斜角或远距离观看纹理时的采样方式
- 可选值为 “None（无）”、“RGSS” 与 “各向异性”
- 默认值取决于图形预设与硬件
- “RGSS”（旋转网格超级采样）是一种基于着色器的过滤方式，在中等性能开销下提升纹理清晰度
  - 在所有硬件上可用
- “各向异性” 是一种基于硬件的过滤方式，提供最佳纹理清晰度
  - 最大各向异性可为 “2x”、“4x”、“8x”，由单独的滑块控制
  - 如硬件不支持，各向异性过滤不会产生效果
  - 这会显著增加显存占用（尤其在 mipmap 层数较高时）

### 物品组件

- 为使所有近战武器类型都可使用，`min_reach`、`max_reach` 与 `hitbox_margin` 已从 `minecraft:piercing_weapon` 与 `minecraft:kinetic_weapon` 中移至新组件 `minecraft:attack_range`

#### 新增 `minecraft:attack_range`

- 使用该物品时启用自定义攻击距离。
- 会覆盖玩家的默认实体交互距离。
- 常规近战距离对生物的计算基于其碰撞箱与目标碰撞箱的距离。本组件通过调整攻击判定箱的最小/最大外扩量来修改这项距离检测。
- 使用 `minecraft:kinetic_weapon` 的生物，其攻击距离以与玩家相同的方式应用。
- 格式：对象，字段如下：
  - `min_range`：浮点数，攻击者与目标间被视为有效的最小距离
    - 默认值：`0.0`，有效范围 `0.0` 至 `64.0`
  - `max_range`：浮点数，攻击者与目标间被视为有效的最大距离
    - 默认值：`3.0`，有效范围 `0.0` 至 `64.0`
  - `hitbox_margin`：浮点数，检测有效命中时应用于目标包围盒的外扩边距
    - 默认值：`0.3`，有效范围 `0.0` 至 `1.0`
  - `mob_factor`：浮点数，当由生物使用该物品时，应用到 `min_range` 与 `max_range` 的倍率
    - 默认值：`1.0`，有效范围 `0.0` 至 `2.0`

### 纹理

- `*.mcmeta` 文件的 `texture` 段新增 `alpha_cutoff_bias` 字段，用于控制 cutout 纹理的 Alpha 偏置
  - 技术上该值无硬性上限，也可为负；但高于 `1.0` 或低于 `-1.0` 毫无意义，因为纹理将分别变为完全不透明或完全透明
  - 默认值为 `0.0`；若纹理在远处会过于透明/纤细，可适当提高；若显得过“实（不透明）”，可适当降低
  - 建议以小步进微调以找到合适的值。例如 `kelp`（海带）纹理使用 `0.1`，避免远处完全透明
  - 仅作用于挖空纹理的低级别 mip，不改变第一级 mip 的 Alpha，对其他类型纹理无效
